Core Databases

核心数据库

宏基因组分析中不可或缺的参考数据库，覆盖核酸序列、蛋白质功能、代谢通路、物种分类与耐药基因等多个维度。

Database

GenBank / NCBI

NCBI 核酸序列综合数据库，包含全基因组、16S rRNA 等参考序列，是宏基因组分析最基础的数据来源。

Database

UniProt

高质量蛋白质序列与功能注释数据库，包含 Swiss-Prot（人工审核）和 TrEMBL（自动注释）两部分。

Database

KEGG

京都基因与基因组百科全书，提供代谢通路（PATHWAY）、基因功能（KO/ORTHOLOGY）和模块（MODULE）注释。

Database

eggNOG

基于直系同源群的功能注释数据库，覆盖从细菌到真核生物的多层级分类，支持 GO、KEGG、COG 注释。

Database

InterPro

整合 Pfam、PROSITE、SMART 等多个蛋白质结构域数据库，提供蛋白质家族和功能位点的统一注释。

Database

Pfam

蛋白质家族数据库，基于隐马尔可夫模型（HMM），广泛用于蛋白质结构域和功能区域的鉴定。

Database

ENA

欧洲核苷酸档案库，存储原始测序数据和组装序列，是提交和获取宏基因组原始数据的主要平台。

Database

MGnify

EBI 宏基因组数据平台，提供公开宏基因组数据的浏览、分析和下载，包含物种分类和功能注释结果。

Database

GTDB

基于基因组系统发育的标准化分类系统，解决了传统分类与基因组进化不一致的问题，广泛用于微生物分类。

Database

JGI IMG/M

联合基因组研究所的微生物基因组与宏基因组整合平台，提供丰富的比较基因组学和宏基因组学分析工具。

Database

SILVA

高质量核糖体 RNA 数据库，提供 16S/18S/23S rRNA 的比对和分类信息，是扩增子分析的金标准参考。

Database

Greengenes2

16S rRNA 参考数据库的更新版本，与 GTDB 分类体系兼容，适用于 QIIME2 等扩增子分析流程。

Database

CARD

综合性抗生素耐药基因数据库，涵盖耐药基因、突变和耐药机制的全面注释信息。

Database

NCBI BioProject / SRA

NCBI 的测序 reads 档案（SRA）和项目管理（BioProject），是获取公开宏基因组原始测序数据的核心来源。

Quality Control

数据质控与预处理

高质量数据是下游分析的基础，这些工具帮助你完成过滤、去接头、去宿主污染等预处理步骤。

Tool

fastp

超快速 FASTQ 预处理工具，集成质控、过滤、去接头、去重等功能，输出 HTML 交互报告，替代 FastQC + Trimmomatic。

Tool

FastQC

经典测序数据质量评估工具，生成碱基质量、GC含量、接头污染等多维度质控报告。

Tool

KneadData

Harvard Huttenhower 实验室开发的宏基因组质控流程，集成 Trimmomatic 和 Bowtie2 去宿主污染。

Tool

DIAMOND

超快速蛋白质序列比对工具，比 BLAST 快 500-20000 倍，适用于宏基因组大规模功能注释。

Tool

BWA / Bowtie2

短序列比对工具，常用于将 reads 比对到参考基因组或宿主基因组以去除污染。

Tool

BBTools

DOE-JGI 开发的工具集，BBduk 用于去接头/去污染，BBNorm 用于数字归一化降低组装复杂度。

Assembly

序列组装

将短 reads 拼接为更长的 contigs，是宏基因组分析的关键步骤。

Tool

MEGAHIT

基于 De Bruijn 图的高效宏基因组组装器，内存占用低，适合大规模宏基因组样本的从头组装。

Tool

metaSPAdes

SPAdes 的宏基因组模式，组装质量高但内存需求大，适合中等复杂度的宏基因组样本。

Tool

Canu / Flye

长读长组装工具，适用于 PacBio / Nanopore 长读长宏基因组数据的组装，可生成更连续的 contigs。

Tool

minimap2

通用序列比对工具，支持长读长和短读长，常用于组装后 reads 回贴评估和 hybrid 组装辅助。

Tool

QUAST

基因组/宏基因组组装质量评估工具，统计 N50、contig 长度分布、基因组覆盖度等关键指标。

Genome Binning

基因组分箱 (Binning)

从组装的 contigs 中恢复单菌基因组（MAG），是宏基因组研究的核心环节。

Tool

MetaBAT2

基于四核苷酸频率和覆盖度的宏基因组分箱工具，速度快、精度高，是最常用的 binning 工具之一。

Tool

CONCOCT

基于高斯混合模型的无监督分箱工具，利用序列组成和多样本覆盖度信息进行 contig 聚类。

Tool

MaxBin2

基于期望最大化算法的分箱工具，利用标记基因和四核苷酸频率进行基因组 bin 的自动提取。

Tool

DAS Tool

分箱结果整合工具，综合多个 binning 算法的结果，提取最优 bin 集合。

Tool

CheckM / CheckM2

MAG 质量评估工具，基于单拷贝标记基因评估完整度（completeness）和污染度（contamination）。

Tool

metaWRAP

综合性宏基因组分箱流程，整合多种 binning 工具，提供 bin 提纯、定量和注释的一站式解决方案。

Taxonomic Profiling

物种分类与组成分析

从 reads 水平鉴定微生物群落的物种组成和相对丰度。

Tool

Kraken2

基于 k-mer 精确匹配的超快速物种分类工具，内存效率高，适合大规模宏基因组 reads 的物种鉴定。

Tool

MetaPhlAn 4

基于标记基因的物种组成分析工具，利用约 100 万个独特标记基因精确估计微生物群落组成。

Tool

Bracken

Kraken2 的补充工具，使用贝叶斯方法重新估计物种丰度，显著提高种级别的丰度估计精度。

Tool

Kaiju

基于蛋白质水平比对的物种分类工具，将 reads 翻译为蛋白质后比对参考数据库，更适合远缘物种鉴定。

Tool

Centrifuge

基于 Burrows-Wheeler 变换的高效分类器，支持同时比对多个参考基因组，内存占用低。

Tool

mOTUs

基于通用单拷贝标记基因的物种定量工具，不依赖完整参考基因组，可估计已知和未知物种的相对丰度。

Functional Annotation

功能注释与基因预测

从组装的 contigs 或 reads 中预测基因并赋予功能注释。

Tool

Prodigal

原核生物基因预测工具，基于动态规划识别开放阅读框（ORF），是宏基因组基因预测的标准工具。

Tool

HUMAnN 3

Huttenhower 实验室的宏基因组功能分析工具，定量通路丰度和基因家族丰度，支持 MetaPhlAn 联合分析。

Tool

eggNOG-mapper

基于 eggNOG 数据库的快速功能注释工具，通过直系同源比对分配 GO、KEGG、COG 等功能注释。

Tool

HMMER (hmmscan)

基于隐马尔可夫模型的蛋白质序列搜索工具，用于比对 Pfam、TIGRFAM 等蛋白质结构域数据库。

Tool

InterProScan

整合多个蛋白质签名数据库的一站式扫描工具，提供全面的功能域注释。

Tool

antiSMASH

次级代谢产物生物合成基因簇预测工具，可自动识别 NRPS、PKS、Terpene 等生物合成基因簇。

Tool

dbCAN2 / dbCAN3

碳水化合物活性酶（CAZymes）自动注释服务器，整合 HMMER、DIAMOND 和 Hotpep 三种方法提高注释精度。

Resistome & MGE

抗性基因与移动遗传元件

识别和分析微生物群落中的抗生素耐药基因及其传播载体。

Tool

RGI

基于 CARD 数据库的耐药基因预测工具，使用蛋白质同源模型和 SNP 模型识别抗生素耐药基因。

Tool

AMR++

基于 MEGARes 数据库的宏基因组耐药基因分析流程，专为 reads 水平的耐药基因定量设计。

Tool

ResFinder

丹麦基因组中心开发的耐药基因识别工具，覆盖获得性耐药基因和点突变引起的耐药性。

Tool

MobileElementFinder

移动遗传元件预测工具，识别质粒、转座子、整合子等与耐药基因传播相关的遗传元件。

Tool

VFDB

毒力因子数据库及分析工具，用于识别细菌病原体相关的毒力基因，辅助评估微生物致病潜力。

Pipelines & Platforms

综合分析流程与平台

一站式分析框架，整合多个工具提供可重复的宏基因组分析流程。

Pipeline

nf-core/mag

基于 Nextflow 的宏基因组 MAG 分析流程，涵盖质控、组装、分箱和质量评估，可重复性好。

Pipeline

ATLAS

基于 Snakemake 的宏基因组一站式分析流程，集成组装、分箱、定量和注释，支持集群和云计算。

Pipeline

bioBakery Workflows

Huttenhower 实验室的分析流程框架，整合 MetaPhlAn + HUMAnN，提供从原始 reads 到功能分析的全流程。

Pipeline

metaWRAP

综合性宏基因组分析工具包，提供 QC、组装、分箱、提纯、定量、注释和可视化的模块化流程。

Pipeline

SqueezeMeta

全流程宏基因组分析管线，从 reads 到物种和功能注释，支持多样本合并组装和 co-assembly 模式。

Pipeline

nf-core/taxprofiler

基于 Nextflow 的分类谱分析流程，可同时运行 Kraken2、MetaPhlAn、Centrifuge 等多个分类器并比较结果。

Pipeline

QIIME 2

最流行的微生物组分析平台之一，提供完整的扩增子分析流程，插件生态丰富，社区活跃。

Pipeline

DADA2

高分辨率扩增子序列变异（ASV）推断算法，替代传统 OTU 聚类，可在单核苷酸分辨率下区分微生物类群。

Pipeline

MG-RAST

在线宏基因组分析服务器，提供自动化的质控、注释和比较分析，适合快速探索性分析。

Visualization & Statistics

可视化与统计分析

将分析结果转化为高质量图表，支持交互式探索与论文出版。

Web Tool

MicrobiomeAnalyst

在线微生物组数据分析平台，支持 16S 和宏基因组数据，提供丰富的统计分析和交互式可视化。

Web Tool

Galaxy

开源生物信息学分析平台，通过 Web 界面提供可重复的分析工作流，内置大量宏基因组分析工具。

R Package

phyloseq

R 语言微生物组分析包，整合 OTU/ASV 表、分类信息和样本元数据，提供 Alpha/Beta 多样性分析和可视化。

Tool

Krona

交互式层级数据可视化工具，生成可缩放、可搜索的 HTML 饼图，广泛用于展示物种分类结果。

Tool

Anvi'o

综合性微生物组可视化与分析平台，支持基因组分箱可视化、比较基因组学和泛基因组分析。

Tool

GraPhlAn

微生物群落组成圆形进化树可视化工具，生成高分辨率分类树图，适合论文出版级别的图片制作。

Tool

Pavian

基于 Shiny 的交互式宏基因组分类结果可视化工具，支持 Kraken2/Centrifuge/MetaPhlAn 输出的探索。

Workflow Overview

典型分析流程概览

四条主流的宏基因组分析工作流，展示从原始数据到最终结果的完整步骤。

宏基因组 Shotgun 分析流程

原始 ReadsIllumina / Nanopore
fastp 质控过滤、去接头
KneadData去宿主污染
MEGAHIT 组装Contigs 生成
Prodigal 基因预测ORF 识别
DIAMOND / eggNOG功能注释
此流程适用于以基因目录（gene catalog）为核心的宏基因组分析策略，侧重于功能基因的挖掘和定量。

MAG 恢复（Genome-Resolved）流程

质控后 Reads
MEGAHIT / metaSPAdes组装
BWA / minimap2Reads 回贴
MetaBAT2 + CONCOCT + MaxBin2多工具分箱
DAS ToolBin 整合
CheckM2质量评估
GTDB-Tk物种鉴定
推荐使用 DAS Tool 整合三种 binning 算法的结果，并仅保留完整度 >90%、污染度 <5% 的高质量 MAG。

扩增子（16S rRNA）分析流程

原始 Reads
fastp / DADA2质控与过滤
DADA2 / DeblurASV 推断
SILVA / Greengenes2分类注释
QIIME 2 / phyloseq多样性分析
PICRUSt2 / Tax4Fun2功能预测

耐药基因组（Resistome）分析流程

质控后 Reads
RGI / AMR++CARD / MEGARes 比对
ResFinder耐药基因鉴定
MGE 分析移动元件关联
丰度定量与可视化耐药谱分析
